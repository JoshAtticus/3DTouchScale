# 3DTouchScale
Simple Scale using 3D Touch.

## How it works
See https://ashertrockman.com/ios/2015/10/24/3d-touch-scale.html

## Supported Devices
- iPhone 6s
- iPhone 6s Plus
- iPhone 7
- iPhone 7 Plus
- iPhone 8 (Tested!)
- iPhone 8 Plus
- iPhone X
- iPhone Xs
- iPhone Xs Max

The iPhone Xr, 11 and later, SE 1/2/3 and so on do not work as they do not have 3D Touch.
